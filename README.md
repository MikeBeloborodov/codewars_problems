## About

Various problems by kyu from website https://www.codewars.com/

### Clone and test

```sh
git pull https://github.com/MikeBeloborodov/codewars_problems
cd codewars_problems
npm i
npm run test
```

### Add new kata
```sh
bash new.sh $kata_folder $kata_name
```

### Refresh kata count
```sh
node count.js
```
